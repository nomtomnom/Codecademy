Why is Comic Sans still a thing

Semantic Element Name Description of Element
<section> Section A grouping of content, typically with a heading.
<article> Article Independent, self-contained content. An article should be able to be reproduced without the rest of the website. Used for forum posts, blogs, newspaper articles.
<header> Header A container for introductory content or navigational links.
<nav> Navigation A block of navigational links for the current website.
<aside> Aside Meant for tangentially related content, akin to a footnote.
<figure> Figure A section for self-contained illustrations, diagrams, and the like. Can contain the <figcaption> element.
